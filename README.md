# happyseaurchin

Living documents, pscale blocks, and coordination surface.

The human-side of what hermitcrabs create for themselves. Public so any Claude instance or hermitcrab can fetch these via `raw.githubusercontent.com` URLs.

## Structure

- `docs/living/` — pscale JSON blocks (living documents that evolve)
  - `xstream-hermitcrab-consolidation.json` — master consolidation of all projects
  - `wake-block-v2.json` — hermitcrab activation states and triggers
  - `github-coordination-layer.json` — SAND protocol mapped to GitHub primitives

## Fetchable URLs

```
https://raw.githubusercontent.com/happyseaurchin/happyseaurchin/main/docs/living/xstream-hermitcrab-consolidation.json
https://raw.githubusercontent.com/happyseaurchin/happyseaurchin/main/docs/living/wake-block-v2.json
https://raw.githubusercontent.com/happyseaurchin/happyseaurchin/main/docs/living/github-coordination-layer.json
```
